# NOCAP: 유튜브 가짜뉴스 판별 확장 프로그램

유튜브 영상에 내재된 다중 양태의 정보(자막, 문맥)를 실시간으로 수집하고, 온디바이스 AI 인프라를 통해 교차 검증하여 직관적인 신뢰도 백분율로 제공하는 크롬 확장 프로그램입니다.

## 🚀 프로젝트 개요 및 아키텍처 (무비용 Freemium 모델)

본 프로젝트는 비용 없이 유지 가능한 **로컬 AI(On-device)** 기반 아키텍처로 설계되었으며, **Freemium(기본 무료, 프리미엄 유료)** 수익 모델을 채택하고 있습니다.

*   **서버리스 & 온디바이스 처리**: 무거운 백엔드 서버 대신 Chrome Built-in AI (Gemini Nano)와 무료 외부 API(Google Fact Check Explorer)를 사용하여 API 유지 비용을 0원으로 수렴합니다.
*   **Shadow DOM 격리**: 확장 프로그램의 UI 요소가 유튜브 본래의 CSS와 충돌하지 않도록 Shadow DOM 내부에 렌더링됩니다.
*   **Freemium 모델**:
    *   **Basic (무료)**: 영상의 종합 신뢰도 점수(%)와 시각적 인디케이터(색상)만 제한적으로 제공.
    *   **Premium (유료)**: 팩트체크 세부 근거, 페널티 요인, 원본 신뢰 문서 링크 등 전체 분석 데이터를 해제(Blur 제거). (Firebase Auth 기반 인증)

## 🛠️ 개발 타스크 밎 진행 상황 (Task Breakdown)

### 1. 초기 설정 및 기반 아키텍처 구성
- [x] Manifest V3 템플릿 프로젝트 생성 (`manifest.json`, `background.js` 등)
- [ ] Webpack/Vite 등 빌드 도구 도입 (선택적이나 모듈화 위해 필요시)
- [ ] Shadow DOM 형태의 UI 인젝션 구조 설계 (스타일 오염 방지)

### 2. UI/UX 컴포넌트 개발 (Shadow DOM 내부) & 결제 연동
- [ ] 신뢰도 게이지 바 컴포넌트 구현 (초록/노랑/빨강 시각화)
- [ ] 베이직 플랜용 블러(Blur) 처리된 디스플레이 및 결제 유도 버튼 로직 구현
- [ ] Firebase Auth 및 Firestore 연동 (`auth.js`) - Premium 플랜 상태 중앙 관리

### 3. 데이터 추출 파이프라인 (콘텐츠 스크립트)
- [ ] 유튜브 동영상 플레이어 내 자막 데이터 실시간 스크래핑 로직 구현 (동적 파싱)
- [ ] 현재 시청 중인 채널의 메타데이터(채널명, URL 등) 수집

### 4. 온디바이스 AI (Zero-Cost) 파이프라인 구축 (`analyzer.js`)
- [ ] Chrome Built-in AI (Gemini Nano) `window.ai` API 연동 지원 여부 확인 및 프롬프트 환경 구성
- [ ] 자막 데이터 기반 원자적 주장 분할(Chunking) 및 감성 분석 로직 구현

### 5. 무비용 클라우드 연동 및 통합 검증 엔진 
- [ ] 로컬 AI가 추출한 주장을 바탕으로 외부 무료 RAG (Google Fact Check API) 서버통신 모듈 개발
- [ ] 다중 가중합 모델($S_{fact}$, $S_{source}$) 구현 및 최종 백분율 산출 로직 적용
- [ ] 산출된 데이터를 바탕으로 상세 근거(Reasoning) 텍스트를 구성하여 UI 단으로 전달

### 6. 권한 및 최적화, 최상위 엣지케이스 대응
- [ ] SPA(Single Page Application) 라우팅 전환 시 구독/자원 해제 처리 로직 (메모리 누수 방지)
- [ ] 데이터 휘발성(Volatile 처리) 원칙을 적용하여 캐싱된 분석 데이터를 메모리에만 보관 후 탭 종료 시 삭제 처리
